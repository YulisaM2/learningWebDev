The Grid System

Each interface is divided into 12 units, we can use them to manage content and display it based on a proportional scale of the screen size.
The 4 layouts for this grid include:
  1. col-xs (phones)
  2. col-sm (tablets)
  3. col-md (medium desktops)
  4. col-lg (large desktops)
