Already tried email, password and text, now we are going to use radio, select and textarea tags

Radio

Once selected can not be unselected, used when user has only one choice

Example

With which hand do you type faster?

Checkbox

Can be unselected, normally used for multiple option questions

Example

What do you like to do on you spare time?

Select & Option

Creates a drop down menu with different options

Textarea

So the user can enter larger text than in an input text element, like a descriptive bio
